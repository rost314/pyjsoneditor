Binary packages have no dependencies.

If you will use the source, you need the dependencies stated below.

### Dependencies ###
- python 2.5 or 2.6 or 2.7 (http://python.org)
Not yet tested on python 2.7 but should work :)

- simplejson 2.1.2 at least [Revision 232](https://code.google.com/p/pyjsoneditor/source/detail?r=232)
(http://code.google.com/p/simplejson/source/checkout)

- ordereddict 0.4.4 (http://www.xs4all.nl/~anthon/Python/ordereddict)

- chardet 2.0.1 (http://chardet.feedparser.org)
Tested only with chardet version 2.0.1, may work with versions prior to 2.0.1

- PyQt4 4.4.4 or newer (http://www.riverbankcomputing.co.uk)
Tested with PyQt4 4.4.4 and 4.7.3, may work with versions prior to 4.4.4

- QScintilla2 (http://www.riverbankcomputing.co.uk/software/qscintilla/download)
Tested with QScintilla2 2.3.2 and 2.4.4, may work with versions prior to 2.3.2
